## Описание проекта

Проект посвящён дообучению русскоязычной визуально-языковой модели (VLM) на открытых данных VK для решения задач визуального вопросно-ответного взаимодействия (VQA) и визуального рассуждения.

### Цель
Получить конкурентоспособную русскоязычную VLM, демонстрирующую высокие метрики на бенчмарках **GQA-ru** (визуальное рассуждение) и **MMBench-ru** (комплексная оценка), путём дообучения базовой модели `Qwen2-VL-2B-Instruct` на открытых датасетах из коллекции `deepvk`.

### Задачи
1. Изучить коллекцию открытых датасетов VK на HuggingFace.
2. Отобрать датасет для обучения VLM (`deepvk/LLaVA-Instruct-ru`).
3. Настроить пайплайн Parameter-Efficient Fine-Tuning (LoRA) в Google Colab.
4. Провести дообучение модели на отфильтрованных данных.
5. Оценить качество модели на бенчмарках `deepvk/GQA-ru` и `deepvk/MMBench-ru`.
6. Задокументировать процесс и результаты.

### Актуальность и жизнеспособность
- Большинство SOTA VLM ориентированы на английский язык.
- Бизнесу нужны русскоязычные решения для анализа изображений (модерация, ритейл, медицина).
- Проект опирается на открытый стек VK и может быть масштабирован.

## Использованные данные VK

Все данные взяты из коллекции [deepvk/vision-language-modeling](https://huggingface.co/collections/deepvk/vision-language-modeling-664dd7e4c257cc78e740f6bc):

| Датасет | Назначение | Объём |
|---|---|---|
| `deepvk/LLaVA-Instruct-ru` | Обучение (SFT) | 200 примеров (подмножество) |
| `deepvk/GQA-ru` | Бенчмарк визуального рассуждения | 50 примеров (testdev) |
| `deepvk/MMBench-ru` | Комплексный бенчмарк VQA | 50 примеров (dev) |

### Как использовались данные
- **LLaVA-Instruct-ru**: диалоги формата `human → gpt` по изображениям COCO 2017. Картинки загружались напрямую с `images.cocodataset.org` по ID.
- **GQA-ru**: разбит на две конфигурации — `testdev_balanced_images` (картинки) и `testdev_balanced_instructions` (вопросы/ответы). Связаны по полю `id` = `imageId`.
- **MMBench-ru**: вопросы с вариантами ответов A/B/C/D и правильным ответом.

## Архитектура решения

### Базовая модель
**Qwen2-VL-2B-Instruct** — 2B параметров, отличная поддержка русского языка, высокое разрешение изображений.

### Метод обучения: LoRA (Low-Rank Adaptation)
- Заморожены веса базы и визуального энкодера.
- Обучаются только низко ранковые матрицы в attention-слоях.
- **Параметры:** `r=8`, `alpha=16`, `target_modules=["q_proj", "v_proj", "k_proj", "o_proj"]`
- **Обучаемые параметры:** 2,179,072 / 2,211,164,672 (**0.0985%** от всех)

### Оптимизации для Google Colab Free
- 4-bit квантизация (`BitsAndBytesConfig`, `nf4`)
- `paged_adamw_8bit` — 8-bit оптимизатор
- `gradient_checkpointing=True`
- `max_length=512`, `per_device_train_batch_size=1`, `gradient_accumulation_steps=4`

- Эпох: 1
- Шагов: 50
- Loss на шаге 20: **6.62**
- Loss на шаге 40: **6.17**

**Ответ модели:** «A. семя...»

**Вопрос:** «Посуда на столе выглядит чистой и черной?»  
**Правильный ответ:** нет  
**Ответ модели:** «На столе нет чистой и черной посуды.»
